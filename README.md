# 学生信息管理系统

一个实用的小型通讯录程序，具有添加、查询和删除功能。由姓名、籍贯、电话号码1、电话号码2、电子邮箱组成，姓名可以由字符和数字混合编码。电话号码由数字组成。

## 实现功能

1. 导入通讯录数据文件
2. 插入新记录功能
3. 信息浏览功能
4. 信息查询功能
5. 信息修改功能
6. 信息删除功能
7. 保存功能
8. 系统退出功能

**注意事项：**
- 注意存在同名用户情况
- 电话号码必须保证是数字的
- 注意1个电话号码只能属于一个用户
